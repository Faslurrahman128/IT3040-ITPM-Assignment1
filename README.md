# IT3040 Assignment 1 - Thanglish to Tamil Translator Tests

## Setup Instructions
1. Install Node.js (version 16 or above)
2. Open terminal/command prompt in this folder
3. Run: `npm install`
4. Install browsers: `npx playwright install`

## Running Tests
- Run all tests: `npm test`
- Run with browser visible: `npm run test:headed`
- Generate HTML report: `npm run test:report`

## Test Structure
- 24 positive functional tests
- 10 negative functional tests
- 1 UI test
Total: 35 automated test cases
